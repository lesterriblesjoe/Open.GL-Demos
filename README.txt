
Open.GL Demos

--------


Suite of code demos "mostly" following the samples and exercises from http://open.gl



To build this project:

git clone https://github.com/danlbarron/Open.GL-Demos Open.GL-Demos
git submodules init
git submodules update


Locate the according solution folder in "Open.GL Demos/build"
Now, build that bad boy ;)



Let me know if you have any suggestions/issues/etc..
